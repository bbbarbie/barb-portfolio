# barbie zhu · personal site

my corner of the internet.

**live:** [barbie-site.vercel.app](https://barbie-site.vercel.app)

## what's inside

- a poster-style index map with a 3D die. click it and let chance decide where you go
- a command palette (`⌘K` or `/`)
- keyboard shortcuts: `D` for dance mode, `F` for my actual nyc food picks, `R` to roll the die
- a few other things you'll have to find yourself

## stack

html, css, vanilla js.

## run locally

open `index.html` in a browser.

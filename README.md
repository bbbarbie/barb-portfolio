# barbie zhu · personal site

my corner of the internet. built as one hand-crafted `index.html`: no framework, no build step, no dependencies.

**live:** [barbie-site.vercel.app](https://barbie-site.vercel.app) *(update this link after deploying)*

## what's inside

- a poster-style index map with a 3D die. click it and let chance decide where you go
- a live prediction market that trades on whether you'll scroll to the end
- a command palette (`⌘K` or `/`)
- keyboard shortcuts: `D` for dance mode, `F` for my actual nyc food picks, `R` to roll the die
- a few other things you'll have to find yourself 🐰

## stack

html, css, vanilla js. that's it. the whole site is one file on purpose.

## run locally

open `index.html` in a browser. done.
